# Mobile App Guide

**Feature:** Wave 7
**Platform:** React Native
**Difficulty:** Intermediate
**Time:** 30 minutes

## Overview
React Native mobile app for iOS and Android with full feature parity to web companion.

## Installation
\`\`\`bash
cd mobile
npm install

# iOS
npx pod-install
npm run ios

# Android
npm run android
\`\`\`

## Features
- All Wave 6-7 features
- Native performance
- Offline-first architecture
- Push notifications
- Camera integration
- Bluetooth connectivity
